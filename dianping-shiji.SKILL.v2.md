# 大众点评食记写作 Skill v2.1

## 触发条件

用户输入店名（如"帮我写XX店的食记"、"XX餐厅"、"XX面馆怎么写"），或明确提到"大众点评食记""探店笔记""美食评价"时自动触发。

---

## 核心原则

### 四不
1. **不幻想**：所有菜品、价格、环境细节必须来自真实评论或公开信息，绝不凭空编造
2. **不夸大**：不用"天花板""封神""绝绝子""此生必吃"等过度吹捧词
3. **不文绉绉**：禁止书面语堆砌、成语连排、排比句轰炸
4. **不AI腔**：禁用"总体而言""值得一提的是""不得不说""众所周知""在…中""不仅…而且"等模板句式

### 四要
1. **要真实**：像饭搭子酒后聊天的语气，有优点也有小遗憾
2. **要白话**：口语化、短句为主（70%≤15字）、段落≤3行
3. **要SEO**：店名完整出现3-4次，区域+路名+地标各至少1次
4. **要有画面**：读完能让读者脑补出"我自己去吃会是什么样"

---

## 写作引擎（核心升级）

### 引擎1：开头钩子（50-80字）

禁止平铺直叙"今天去了XX店"。必须从以下三种钩子类型中选一种：

**类型A·场景痛点钩子**
> 加班到九点从塔园路地铁站出来，肚子叫得整栋写字楼都听见了。抬头看见易生活广场还亮着灯，斗叁千门口的大台阶像在跟我招手。

**类型B·反常识钩子**
> 人均120的土菜馆开在商场一楼，工作日晚上还要排号，这事本身就有点反常。

**类型C·感官暴击钩子**
> 推开斗叁千的门，先闻到一股红烧肉混着豆香的复合味，紧接着是热毛巾递过来的温度。那一刻我知道，今晚来对了。

### 引擎2：五感颗粒度（每道菜必须覆盖至少3个感官维度）

每道菜从以下五个维度中选取至少3个来写，**禁止只用"好吃/嫩/鲜/香"等空洞形容词**：

| 维度 | 错误写法 | 正确写法 |
|------|---------|---------|
| 视觉 | 卖相很好 | 端上来的时候油还在跳，整条鱼浸在红亮的酱汁里 |
| 嗅觉 | 很香 | 凑近先闻到豆豉的发酵味，然后才是鱼的鲜甜 |
| 触感/温度 | 很嫩 | 筷子一拨肉就散开了，入口是烫的，得吹两下 |
| 味觉层次 | 味道不错 | 第一口偏甜，嚼到第三下咸味才出来，最后回口有点辣 |
| 听觉 | / | 端上桌的时候铁板还在滋滋响，隔壁桌都扭头看了一眼 |

**每道菜至少包含1个比较级**（比XX更XX/没有XX那么XX/跟XX不太一样），制造对比感。

### 引擎3：饭搭子人设

全文用"聊天体"而非"说明书体"。具体手段：

- **夹带私货**：「我跟你们说」「说实话」「怎么说呢」这类口语词自然穿插
- **暴露小缺点**：每个店至少提1个无关痛痒的小槽点（分量偏小/某个菜一般/服务偶有掉线/停车不太方便），制造真实感
- **埋互动钩子**：结尾加一句「你们要是去的话，记得先XXX，别问我怎么知道的」或「偷偷说一句，XXX比XXX好吃多了」
- **"偷偷告诉你"句式至少用1次**

### 引擎4：环境空间叙事（100-150字）

禁止"装修不错""环境很好"这类空话。用动线叙事法，把读者从门口带进去：

**必含三个层次**（按顺序写）：

1. **门面第一眼**（20-30字）：门口长什么样、招牌风格、进门的第一感受
2. **内部一个抓人细节**（30-50字）：不是面面俱到，挑一个最特别的——墙上挂的画、灯光的颜色、桌椅的质感、某个装饰物。**必须具体到某一件东西**，不能说"中式装修"要说"墙上挂着叶圣陶的字画，裱的是原木框"
3. **一个服务瞬间**（20-30字）：不是"服务员很热情"，要写具体动作——递毛巾的速度、倒茶的手法、引座时的语气。用画面代替评价

**附加技巧**：
- 用对比制造反差：大厅vs包厢、白天vs晚上、工作日vs周末
- 挖掘1个"拍照点"：哪个角落/哪个角度是出片的，带一句

错误写法：
> 环境装修偏中式，整体很有格调，包厢比大厅更有档次。

正确写法：
> 门口的大台阶先给你一个下马威，走上去才看到斗叁千的招牌。推门进去，最先注意到的是墙上叶圣陶的字画，原木框裱的，不是那种打印店做的廉价货。大厅的灯光偏暖黄，桌子之间距离够宽，不会听见隔壁聊什么。包厢更安静些，每个包间门上都挂了木牌，一股旧时私塾的味道。服务员领位的时候顺手就把椅子拉开了，没等你动手。

### 引擎5：反差记忆点

从搜索素材中挖掘至少1个反差点，嵌入文中。常见反差类型：

| 类型 | 示例 |
|------|------|
| 环境vs价格 | 门头像高端会所，翻开菜单人均才80 |
| 外表vs味道 | 卖相一般甚至有点丑，但一口下去直接闭嘴了 |
| 人设反差 | 纹身花臂的老板端出来的甜品精致得像法餐 |
| 预期落差 | 冲着招牌菜来的，结果被一个凉菜圈粉了 |
| 地域反差 | 在苏州吃到了比湖南还地道的剁椒鱼头 |

## 写作流程

### 第一步：搜集真实信息

用 web_search 搜索（并行发起，至少3条）：
- `{店名} 大众点评` → 评分、人均、推荐菜
- `{店名} 怎么样` → 真实食客评价原文
- `{店名} 地址 怎么走` → 到达方式和周边地标

### 第二步：提取关键素材

从搜索结果中必提取：
- 3-5个真实菜品名 + 价格区间
- 2-3条真实评论中的具体感受原话（用于改写，不是照抄）
- 到达方式（地铁/公交 + 步行路线 + 显眼参照物）
- 周边地标（商场/学校/写字楼/景点）
- **至少1个反差点**（找不到就翻评论找槽点或意外好评）

### 第三步：按新结构写作（500-700字）

---

**【标题】（15-20字）**

格式：主体标识 + 价值亮点 + 情绪触发词

示例：
- 塔园路这家土菜馆，人均120吃出了请客的排面
- 藏在购物广场里的斗叁千，鱼头汤鲜到我多喝了两碗

---

**【开头】场景钩子（50-80字）**

三选一：场景痛点 / 反常识 / 感官暴击

---

**【环境】空间叙事（100-150字）**

按"门面第一眼→内部抓人细节→服务瞬间"三层写。必须具体到某一件东西（墙上的画/桌椅质感/灯光）。见引擎4。

---

**【菜品1】第一个记忆点（120-150字）**

- 菜名直接说，不引号
- 至少3个感官维度（视觉/嗅觉/触感/味觉层次/听觉）
- 至少1个比较级
- 价格自然带出

---

**【菜品2】另一个角度（100-130字）**

与菜品1拉开差异（一个是汤/一个是主食/一个是荤/一个是素），避免重复感。同样至少3个感官维度。

---

**【菜品3】（可选，80-100字）**

如果食材丰富可加一个菜。否则用一个"意外发现"或"小槽点"来填补。

---

**【到达+周边】（50-70字）**

地铁X号线XX站X口，出来怎么走，看到什么就到了。附近可以顺便逛XXX。

---

**【收尾】记忆切片（30-50字）**

不要总结，不要"推荐/不推荐"。用一个让读者记住的画面或感受收尾。

错误：总的来说这家店性价比很高，值得推荐。
正确：走到地铁站的路上还在咂摸那个鱼汤的鲜，这大概就是吃完斗叁千的正常反应。

---

## 语言风格细则

### 句式要求
- 短句占70%以上（≤15字）
- 禁止连续3句以上相同句式
- 偶尔用"就是""反正""说实话""怎么说呢"破节奏

### 禁用词清单
**AI味禁用词**：总体而言、值得一提的是、不得不说、众所周知、在…中、不仅…而且、既…又、通过…实现、总而言之、综上所述

**浮夸禁用词**：天花板、封神、绝绝子、此生必吃、惊艳、灵魂、治愈、宝藏、入口即化、YYDS、yyds、绝了、哭了

**书面禁用词**：饕客、珍馐、唇齿留香、回味无穷、食指大动、饕餮盛宴

**空洞形容词（除非紧跟感官细节，否则禁用）**：好吃、美味、很棒、不错、还行

### 标点规则
- 句号为主，感叹号全文≤2个
- 不用分号
- 引用食客原话用「」不用""
- 口语停顿用逗号，不要用破折号

### 必含SEO要素
- 店名完整出现3-4次
- 区域名（虎丘区/朝阳区等）+ 路名
- 最近地铁站名
- 1个周边地标
- 2-3个菜品关键词

## 输出前自检

完成初稿后，逐条检查：
1. [ ] 有没有编造没搜到的菜品或价格？
2. [ ] 有没有出现禁用词？
3. [ ] 开头是钩子还是平铺直叙？
4. [ ] 每道菜覆盖了几种感官维度？（至少3种）
5. [ ] 有没有至少1个比较级？
6. [ ] 有没有至少1个反差点或小槽点？
7. [ ] 环境描写有没有具体到某一件东西？（不能只写"中式装修"）
8. [ ] 环境有没有按"门面→细节→服务"三层写？
9. [ ] 有没有"饭搭子"聊天感？
10. [ ] 读起来像不像真人吃完写的？
11. [ ] 字数在500-700之间？
12. [ ] SEO要素都埋进去了吗？

## 使用示例

**用户输入**：「老北京炸酱面大王（崇文门店）」

**Skill 执行**：
1. 并行搜索3条关键词
2. 提取素材（菜品+价格+真实评价+反差点+到达方式+周边）
3. 选钩子类型 → 五感写菜品 → 夹带槽点 → 记忆切片收尾
4. 自检10条 → 输出

## 附：快速诊断表

如果写出来的食记"不够吸引人"，对照此表排查：

| 症状 | 病因 | 解法 |
|------|------|------|
| 读完不想去 | 缺少感官细节 | 加视觉/嗅觉/触感描写 |
| 开头划走 | 没有钩子 | 换场景痛点/反常识钩子 |
| 像AI写的 | 缺少人味 | 加槽点、口语词、"偷偷告诉你" |
| 记不住 | 没有反差 | 挖掘环境vs价格/预期vs实际 |
| 看不完 | 段落太长 | 每段不超过3行 |
| 没画面 | 只有味觉 | 补听觉（滋滋声）、触感（烫/脆/弹） |
| 环境没印象 | 空话"装修不错" | 具体到一件东西，按门面→细节→服务三层写 |
